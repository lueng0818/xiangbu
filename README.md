\# Tilandky 媽媽覺察陪伴室 (Landing Page)



這是一個專為忙碌媽媽設計的線上覺察落地頁，結合了工程師邏輯與身心靈療癒。



\## 功能特色

\- \*\*數據視覺化\*\*：展示媽媽精力分配與核心痛點。

\- \*\*TRUST 系統展示\*\*：清晰呈現 6 個月的轉化流程。

\- \*\*行動呼籲\*\*：引導至 Line 預約諮詢。



\## 如何執行

1\. 安裝套件：`pip install -r requirements.txt`

2\. 執行程式：`streamlit run app.py`
